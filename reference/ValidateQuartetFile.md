# Validate filenames

Verifies that file parameters are character strings describing files
that exist

## Usage

``` r
ValidateQuartetFile(file)
```

## Arguments

- file:

  Variable to validate

## Value

`TRUE` if `file` is a character vector of length one describing a file
that exists, a fatal error otherwise.
